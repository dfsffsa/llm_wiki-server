## 关键概念（30 秒）

```text
raw/sources/  ──Ingest（LLM）──►  wiki/  ──搜索/Chat──►  用户
                      │                              ▲
                      └── reindex --vectors ──► .llm-wiki/lancedb/
```

| 术语 | 含义 |
|------|------|
| **Ingest** | LLM 把原始材料编译成结构化 `wiki/` 页面 |
| **Embedding** | `reindex --vectors` 写入 LanceDB，供混合搜索（HTTP 搜索目前以关键词为主） |
| **HTTP 服务** | `llm-wiki-server`：静态 UI + REST + Chat SSE |
| **Lite** | 轻量静态问答页，路径 `/lite/` |

## 常用命令速查

```bash
# 构建
./scripts/build-all.sh
VITE_BACKEND=http VITE_API_TOKEN=your-token ./scripts/build-web.sh

# 入库
export LLM_WIKI_PROJECT=~/overseas-github/llm_wiki_projects/ParentingBooks
export LLM_WIKI_CONFIG=overlay/config/server.minimax.local.json
./scripts/llm-wiki ingest "$LLM_WIKI_PROJECT/raw/sources/某章.md" --project "$LLM_WIKI_PROJECT" --config "$LLM_WIKI_CONFIG"
./scripts/ingest-batch.sh

# 向量索引
./scripts/llm-wiki reindex --vectors --project "$LLM_WIKI_PROJECT" --config "$LLM_WIKI_CONFIG"

# 本地 HTTP 服务
export LLM_WIKI_STATIC=upstream/dist
export LLM_WIKI_CONFIG=overlay/config/server.minimax.local.json
export LLM_WIKI_API_TOKEN=your-token
./overlay/server/target/release/llm-wiki-server --bind 127.0.0.1:8080
```
